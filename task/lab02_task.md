# Lab 02 — Классификация оттока клиентов банка
# Lab 02 — Bank Customer Churn Classification

---

##  Описание задания

### Цель работы
Разработать и оценить классификатор, прогнозирующий вероятность прекращения сотрудничества клиента с банком, используя метрики F1 и ROC-AUC.

### Задачи
1. Провести анализ данных (EDA).
2. Проверить корректность типов данных и обработать пропуски.
3. Закодировать категориальные признаки.
4. Подготовить признаки для обучения модели.
5. Разделить данные на обучающую и тестовую выборки.
6. Построить классификатор, добившись **F1 ≥ 0.59** на тестовой выборке.
7. Проанализировать качество модели с помощью **ROC-AUC**.
8. Сделать выводы о значимости признаков и работе модели.

### Требования к результату
- Наличие EDA и комментариев к каждому этапу.
- Обоснование выбора моделей и метрик.
- Выводы по проделанной работе.

---

##  Task description

### Goal
Develop and evaluate a classifier predicting customer churn in a bank using F1 score and ROC-AUC.

### Tasks
1. Perform exploratory data analysis (EDA).
2. Validate data types and handle missing values.
3. Encode categorical features.
4. Prepare features for model training.
5. Split data into training and test sets.
6. Build a classifier achieving **F1 ≥ 0.59** on the test set.
7. Analyze model quality using **ROC-AUC**.
8. Draw conclusions about feature importance and model performance.

### Expected results
- EDA with explanations and comments.
- Justification of model and metric choices.
- Clear conclusions based on results.
