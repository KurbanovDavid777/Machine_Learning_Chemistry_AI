# Lab 01 — Предобработка данных и анализ признаков
## Lab 01 — Data preprocessing and feature analysis

---

##  Описание задания

### Цель работы
Изучить основные этапы предобработки данных, анализа признаков и построения модели машинного обучения для задачи оценки стоимости недвижимости.

### Задачи
1. Проверить корректность типов данных.
2. Обработать пропущенные значения.
3. Найти и удалить дубликаты.
4. Обработать аномальные значения и выбросы.
5. Закодировать категориальные признаки.
6. Выполнить масштабирование или нормализацию данных.
7. Сгенерировать новые признаки и обосновать их выбор.
8. Оценить влияние признаков на стоимость квартиры или цену за квадратный метр.
9. Отобрать значимые признаки и обосновать выбор.
10. Попробовать выделить группы объектов, внутри которых влияние признаков на цену различается.
11. Построить и обучить модель, предсказывающую:
   - стоимость квартиры,
   - цену за квадратный метр,
   - или ценовую категорию (дешёвая / средняя / дорогая).

### Требования к результату
- Наличие EDA и комментариев к каждому этапу.
- Обоснование всех применённых методов.
- Выводы по проделанной работе.

---

##  Task description

### Goal
To study data preprocessing techniques, feature analysis, and machine learning model construction for real estate price prediction.

### Tasks
1. Validate data types.
2. Handle missing values.
3. Detect and remove duplicates.
4. Process outliers and anomalous values.
5. Encode categorical features.
6. Apply feature scaling or normalization.
7. Engineer new features and justify their relevance.
8. Analyze feature influence on apartment price or price per square meter.
9. Select significant features and justify the selection.
10. Identify groups of objects where feature influence differs.
11. Build and train a model that predicts:
   - apartment price,
   - price per square meter,
   - or price category (cheap / medium / expensive).

### Expected results
- Exploratory data analysis with explanations.
- Justification of all preprocessing steps.
- Clear conclusions based on the results.
