# Lab 03 — Кластеризация стран для благотворительной помощи
# Lab 03 — Clustering Countries for Charity Assistance

---

##  Описание задания

### Цель работы
Проанализировать международные данные о странах и определить, какие страны нуждаются в помощи в первую очередь, используя методы кластеризации.

### Задачи
1. Провести анализ данных (EDA).
2. Предобработать данные (масштабирование, нормализация, пропуски).
3. Применить следующие методы кластеризации:
   - K-Means / MiniBatch K-Means
   - Иерархическая кластеризация
   - DBSCAN
4. Определить оптимальное количество кластеров для каждого метода (построить графики).
5. Сделать выводы по кластерам: определить страны, которым нужна помощь в первую очередь.
6. Описать эталонную страну в каждом кластере.

### Требования к результату
- EDA с комментариями.
- Визуализация кластеров.
- Обоснование выбора количества кластеров.
- Выводы по странам и кластерам.

---

##  Task description

### Goal
Analyze international country data to identify which countries need assistance first, using clustering methods.

### Tasks
1. Perform exploratory data analysis (EDA).
2. Preprocess data (scaling, normalization, handle missing values).
3. Apply the following clustering methods:
   - K-Means / MiniBatch K-Means
   - Hierarchical clustering
   - DBSCAN
4. Determine the optimal number of clusters for each method (plot graphs).
5. Draw conclusions by clusters: identify countries needing assistance first.
6. Describe a reference country for each cluster.

### Expected results
- EDA with explanations.
- Cluster visualizations.
- Justification for the number of clusters chosen.
- Conclusions about countries and clusters.
