# Lab 05 — Прогнозирование временных рядов
# Lab 05 — Time Series Forecasting

---

##  Описание задания

### Цель работы
Проанализировать сезонные временные ряды по числу сотрудников в индустрии гостеприимства штата Калифорния и построить прогноз на несколько периодов с оценкой точности.

### Задачи
1. Загрузить данные и провести предварительный анализ (EDA).
2. Визуализировать временные ряды и определить тренд, сезонность и возможные аномалии.
3. Проверить стационарность ряда с помощью тестов.
4. При необходимости выполнить преобразования для стационарности.
5. Построить и обучить модель прогнозирования временных рядов.
6. Сделать прогноз на несколько периодов.
7. Оценить точность прогноза с помощью метрик **MAE** и **RMSE**.
8. Визуализировать прогноз вместе с исходным рядом.
9. Проанализировать выявленные сезонные и трендовые компоненты.
10. Обсудить влияние сезонности на прогноз.

### Требования к результату
- Графики временных рядов и прогноза.
- Описание тренда и сезонности.
- Метрики точности прогноза.
- Выводы по работе.

##  Task description

### Goal
Analyze seasonal time series of employees in California’s hospitality industry and build forecasts for several periods with accuracy evaluation.

### Tasks
1. Load the data and perform exploratory data analysis (EDA).
2. Visualize time series and identify trend, seasonality, and anomalies.
3. Test for stationarity.
4. Apply transformations if necessary to achieve stationarity.
5. Build and train a time series forecasting model.
6. Forecast for several periods.
7. Evaluate forecast accuracy using **MAE** and **RMSE**.
8. Visualize the forecast together with the original series.
9. Analyze identified trend and seasonal components.
10. Discuss the impact of seasonality on the forecast.

### Expected results
- Time series and forecast plots.
- Description of trend and seasonality.
- Forecast accuracy metrics.
- Clear conclusions based on the results.
